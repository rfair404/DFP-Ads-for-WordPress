The DFP Complete plugin provides a simple interface for adding DFP Advertisments to your WordPress website or blog. 
== Description ==

Adding Advertising code to your site once required editing template files. Now you can simply add DFP ad tags by copying the DFP head tag and configuring the ad placements. 

== Installation ==

Follow the steps below to install and configure DFP Complete 

1. Upload `dfp-complete` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste the DFP Head code (provided by DFP) into the *DFP Complete* settings page
== Frequently Asked Questions ==

= Do I have to edit any code to use this plugin? =

No, DFP Complet was written so that it *just works* without having to modify any theme code or hooks.

= Can I extend this plugin to do "so and so"?  =

Prollably not. This is the draft release. While there is some extensability built in, there isn't a public api, just yet.
